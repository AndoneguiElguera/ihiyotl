# IHIYOTL - sitio para GitHub Pages

Sitio estático de IHIYOTL: consultoría en biología computacional, bioinformática, multi-ómica y ciencia de datos biomédicos.

## Archivos incluidos

- `index.html`: página principal lista para GitHub Pages.
- `assets/IHIYOTL.jpeg`: logo.
- `assets/favicon.png`: ícono del sitio.
- `assets/Portafolio_IHIYOTL_servicios_bioinformatica.pdf`: portafolio descargable.
- `.nojekyll`: evita procesamiento por Jekyll en GitHub Pages.

## Cómo publicarlo en GitHub Pages

1. Crea un repositorio nuevo en GitHub, por ejemplo: `ihiyotl`.
2. Sube todos los archivos de esta carpeta a la raíz del repositorio.
3. En GitHub, entra a `Settings > Pages`.
4. En `Build and deployment`, elige:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Guarda. En unos minutos GitHub mostrará la URL pública.

## Personalización rápida

Edita en `index.html`:

- correo en los enlaces `mailto:`;
- URLs de LinkedIn, GitHub y ORCID;
- rangos de inversión;
- publicaciones destacadas;
- texto del hero o llamados a acción.

## Nota

El sitio está hecho sin frameworks externos para que cargue rápido y no dependa de paquetes, builds ni rituales digitales innecesarios.
